# Job Application Tracker (Full Stack)

A clean, modern Job Application Tracker to manage applications with a dashboard, filters, and CRUD.

## Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express
- Database: SQLite (file-based)

## Monorepo Structure
job-tracker/
  client/
  server/
  README.md

## Phases (Daily Contributions)
- Day 0: Repo setup + README skeleton + .gitignore
- Day 1: Client setup (Vite+Tailwind) + layout + pages + mock data rendered (Dashboard + Applications)
- Day 2: Client CRUD in local state + Add/Edit modal + filters + status badges
- Day 3: Server setup (Express+SQLite) + DB init + CRUD API routes
- Day 4: Connect client to server (replace mock data) + loading/error states
- Day 5: Dashboard stats from real data + UI polish + empty states + confirm delete
- Day 6: Minimal tests + README final + final cleanup + deployment notes (optional)
